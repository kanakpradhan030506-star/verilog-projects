# RISC-V Instruction Decoder

## Overview

This project implements RISC-V instruction decoder for R-Type instructions.

## Features

- 32 bit instruction input
- Extracts opcode
- Extracts rs1
- Extracts rs2
- Extracts rd
- Extracts funct3
- Extracts funct7
