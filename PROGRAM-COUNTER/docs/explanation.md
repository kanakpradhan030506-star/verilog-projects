# PROGRAM COUNTER 
A Program Counter (PC) is a register in a processor that stores the address of the next instruction to be executed.

# INPUTS 
[7:0] means there will be 8 inputs.

# OUTPUTS
program counter(pc) which will be in increment mode after every positive clk.

# FUNCTIONS 
Functions of Program Counter:
1.Stores current instruction address.
2.Increments after every clock cycle.
3.Resets to 0 when reset is active.
4.Helps the processor fetch instructions sequentially.
