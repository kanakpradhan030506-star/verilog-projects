## Sample FSM Projects

### Traffic Light Controller
Finite State Machine controlling Red, Yellow, and Green traffic signals.

### Vending Machine
FSM-based vending machine that accepts coins and dispenses products.

### Washing Machine Controller
FSM-based controller with the following states:

IDLE → FILL → WASH → RINSE → SPIN → OVER

The controller automatically operates:
- Water Valve
- Wash Motor
- Spin Motor
- Completion Indicator
